
# User Management API

API REST para gerenciamento de usuários desenvolvida com Spring Boot.

Este projeto foi criado como parte dos meus estudos em desenvolvimento backend utilizando Java e Spring Boot, com foco na construção de uma arquitetura de API limpa, incluindo validações, tratamento de exceções e integração com banco de dados.

---

## 🚀 Tecnologias

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Jakarta Validation

---

## 📌 Funcionalidades

- Criar usuários
- Listar usuários
- Buscar usuário por ID
- Atualizar dados do usuário
- Soft delete (desativar usuário)
- Validação de unicidade de e-mail
- Tratamento global de exceções
- Validação de dados de entrada
