# User Management API

REST API for user management built with Spring Boot.

This project was created as part of my backend development studies using Java and Spring Boot, focusing on building a clean API architecture with validation, exception handling, and database integration.

---

## 🚀 Technologies

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Jakarta Validation

---

## 📌 Features

- Create users
- List users
- Find user by ID
- Update user data
- Soft delete (deactivate user)
- Email uniqueness validation
- Global exception handling
- Input validation
